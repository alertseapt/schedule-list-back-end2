Como usar esta página

Como buscar agendamentos

  • Chave de NFe: Digite 44 ou mais dígitos (espaços e letras serão removidos automaticamente)
  • Número da NFe: Digite menos de 44 dígitos para buscar pelo número

Informações exibidas

  • Cliente: Nome, número e CNPJ do estoque selecionado pelo usuário
  • Chave NFe: Clique no campo para selecionar e copiar
  • Produtos: Clique na quantidade para ver lista detalhada
  • Status: Situação atual do agendamento

Ações disponíveis

  • Em Conferência: Botão para alterar status do agendamento
  • Ver Produtos: Modal com detalhes de todos os produtos
  • Copiar Chave: Campo clicável para selecionar a chave NFe
